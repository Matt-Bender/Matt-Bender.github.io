The Unstable Sneak

2D stealth game

The Unstable Sneak is a 2d stealth puzzle game created in Unity. Includes 5 different mazes each of them created initially using random generation to make each unique and unpredictable. It also includes a basic skill system, allowing the player to ramp up in power throughout the game, as well as giving the player additional agency.
